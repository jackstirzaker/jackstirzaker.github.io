# Webfonts

RightFirst webfonts. Comes in `woff2`, `woff` and `variable` formats.


## Use on the web/Tailwind
Add this code to the base `tailwind.css` file and then build the main CSS file.

`@import url('https://repo.rightfirst.co.uk/fonts/fonts.css');`

Then add the fonts to the `tailwind.config.js` file.


### Note on variable fonts 
The variable fonts may not be supported in some browsers. Tailwind will use the static fonts as a fallback.
